# 四声体配置与连接规则

## 声部

上から：Soprano (Sop) → Alto (Alt) → Tenor (Ten) → Bass (Bas)

- **外声**：Sop + Bas（最重要对位关系）
- **内声**：Alt + Ten
- **上3声**：Sop + Alt + Ten

## 各声部音域

Sop: c¹ ~ a²（点线至c³）
Alt: g ~ f²（点线至a²）
Ten: c ~ c²（点线至f¹）
Bas: F ~ c¹（点线至e¹）

## 邻接声部标准距离

- **密集配分**：上3声中最高与最低在8度以内
- **开放配分**：上3声中最高与最低超过8度
- **离隔**：邻接声部超过15度（应避免）
- **Oct配分**：Sop与Alt或Alt与Ten之间为8度

## 上3声标准配置

各3和音的上3声有4种标准配置：
1. 1高, 3, 5（根音最高）
2. 3高, 1, 5
3. 5高, 1, 3
4. 3高, 5, 1（仅I, IV, V的3和音）

## 连接基本法

### 共同音保持
前后和音的共同音尽量在同一声部保持不动。

### 3声进行
- **顺次进行**：2度上下
- **跳越进行**：3度以上
- **3声法则**：上3声中每个声部各自独立进行，禁止2声部同向同度到达同一音。

### 限定进行音
- **导音**（第7音=第3音上方半音）：必须2度上行到主音
- **第7音**（7和音的第7构成音）：必须2度下行
- 导音重复原则上禁止（`伍7` $\rightarrow$ `壹` 时尤为严格）

## 禁止事项

### 连续8度/5度
- 2声部同向进入完全8度或完全5度
- 上2声间尤其严格
- 外声间最严格

### 隐伏8度/5度
- 外声同向进入完全8度/5度（Sop跳越进行时）

### 并达8度/5度
- 2声部从不同音同向到达同一音（同度）

## 容许事项

- 减5度→完全5度（限定进行音形成的连续5度）
- 反向连续5度/8度
- 斜向连续5度/8度
- 内声间连续5度（较外声宽松）
- 配置变化引起的间接连续5度/8度

## 配置变化

和音音度不变仅配置变化（含形体/低音位变化）。

### D和音形体高次化
3和音→7和音→9和音：无条件可能。

### D和音形体低次化
- 9→3：禁止
- 9→7：容许
- 7→3：禁止

### 配置变化中的禁止
- 第7音→根音的2度上行
- 第9音→第3音的2度上行

### 配置变化中的5度/8度
- 配置变化同时生发的连续5度：先行5度为减5度时，后续完全5度容许
- 配置变化引起的并达5度/8度：两外声间全部良好
