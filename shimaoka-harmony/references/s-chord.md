# S和音与变终止

## S和音（`肆`上和音）

### 5种形体

| 形体 | 构成 |
|------|------|
| `肆`（3和音） | 根音+3+5 |
| `肆7` | 根音+3+5+7 |
| `省肆7` | 3+5+7（根省属） |
| `肆9` | 根音+3+5+7+9 |
| `省肆9` | 3+5+7+9（根省属） |

### S和音配置
S和音配置规则与D和音相同。

### S和音连接
S $\rightarrow$ T（变终止）：`肆` $\rightarrow$ `壹`，两和音上3声配分一致。

## 变终止

### 基本型
S $\rightarrow$ T：`肆` $\rightarrow$ `壹`

### 两外声定型
Sop：`肆` $\rightarrow$ `壹`（同度或下行）
Bas：`肆` $\rightarrow$ `壹`（下行4度）

### 上3声配分
两和音配分一致（密集 $\rightarrow$ 密集 或 开放 $\rightarrow$ 开放）。

### 变终止的补充用法
变终止常附加在全终止之后（作为补充终止）。

## S定型

### `肆` $\rightarrow$ `壹`（S $\rightarrow$ T）
基本S定型。上3声配分一致。

### 副S（副降贰和音）
副S：`降贰` 等变位和音。

### S进行
S圈（左回）：`壹` $\rightarrow$ `肆` $\rightarrow$ `柒` $\rightarrow$ `叁` $\rightarrow$ `陆` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹`

## S和音的连用
S和音可连用形成S连用。

### 常用S连用
- `肆` $\rightarrow$ `肆7` $\rightarrow$ `壹`
- `肆` $\rightarrow$ `降贰` $\rightarrow$ `壹`

## S和音的变位

### `肆`上方变位
`升肆` 系列和音。

### `肆`下方变位
`降肆` 系列和音（副 `肆` 系列）。

## S和音在终止中的应用

### 变终止（曲尾补充）
全终止 $\rightarrow$ `肆` $\rightarrow$ `壹`

### 变终止独立使用
较少见，多用于特定风格（如格里高利终止 = Phrygian终止）。

### 弗里吉亚终止
半终止的一种：`肆` $\rightarrow$ `伍`（S $\rightarrow$ D）。Bas下行半音，Sop上行全音。

### 变终止+全终止
变终止后接全终止形成复合终止：`肆` $\rightarrow$ `壹` $\rightarrow$ `伍7` $\rightarrow$ `壹`
