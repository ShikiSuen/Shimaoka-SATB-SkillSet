# SATB 四声体写作示例

下列示例展示从曳动单位选择到具体四声部实现的全流程，可作为 LLM 执行「四声体写作步骤」时的具体参考。

## 示例一：C dur 终止乐句

### 课题设定
- 调性：C dur（长调）
- 长度：4 小节（一个完整曳动单位）
- 要求：以全终止（闭终止）结束

---

### Step 1 & 2: 调性确定 + 曳动单位选择
选择最标准的 K2 型曳动单位：
**T → D2 → D → T** = `壹` → `贰` → `伍` → `壹`

### Step 3: 终止定型确定
- Sop 定型：(1) — `贰` `伍7` `壹`（基本形）
- Bas 定型：(a) — `贰` → `伍` → `壹`（上行 4 度 → 下行 4 度）
- D2项：`贰`〈基〉

### Step 4 & 5: Bas 线条 + Sop 旋律

```
Sop: c'' - a'  - b'  - c''
Alt: g'  - f#' - g'  - e'
Ten: e'  - d'  - d'  - c'
Bas: C   - F   - G   - C
```

### 和音表示

| 拍位 | 和音 | 低音位 | 备注 |
|------|------|--------|------|
| 1 | `壹`〈基〉| 原位 | T，起点，两外声 8 度（密集配分） |
| 2 | `贰7`〈基〉| 原位 | D2，Alt 导音 f#'（升4度） |
| 3 | `伍7`〈基〉| 原位 | D，Ten 第 7 音 d' → c'（2 度下行） |
| 4 | `壹`〈基〉| 原位 | T，终点（全终止），闭终止 |

---

### 规则检查还原

| 规则条款 | 检查内容 | 结果 |
|----------|----------|------|
| B2 | 导音 f#'（Alt，第 3 拍）→ g'（Alt，第 4 拍）：2 度上行 | ✓ |
| B2 | 第 7 音 d'（Ten，第 3 拍）→ c'（Ten，第 4 拍）：2 度下行 | ✓ |
| C1 | 平行 8 度/5 度检查 | ✓ 无 |
| C3 | 并达 8 度/5 度检查 | ✓ 无 |
| C5 | 声部交叉 (Sop > Alt > Ten > Bas) | ✓ 无 |
| A1 | 第 5 音省略 | ✓ 未省略 |
| 共同音保持 | 第 2→3 拍：Alt f#'→g'（2 度上行），Ten d'→d'（保持）| ✓ |

## 示例二：同一乐句的 D 定型表形式

使用 `notation-syntax.md` 的纯文本记号书写：
- 进行：`壹` → `贰7` → `伍7` → `壹`（K2 全终止）
- Sop 定型：(1) `贰` `伍7` `壹`
- Bas 定型：(a) `贰` → `伍` → `壹`（上 4 → 下 4）

四声部（C dur）：
```
Sop: 壹(1高) → 贰7(3高) → 伍7(5高) → 壹(1高)
Alt: 壹(5)   → 贰7(1)   → 伍7(3)   → 壹(3)
Ten: 壹(3)   → 贰7(7)   → 伍7(1)   → 壹(5)
Bas: 壹(1)   → 贰7(1)   → 伍7(1)   → 壹(1)
```

> 说明：上述「(1高)(3高)(5高)(7高)」指该和音中 Sop 所担任的构成音身份；
> 「(1)(3)(5)(7)」指 Bas 对应担任的构成音身份。例如 `贰7(3高)` 表示在 `贰7` 和音中，
> Sop 唱第 3 音（f#'），Bas 唱根音（F）。

## 示例三：C dur における保续低音

### 课题
`壹` 在 Bas 持续 4 拍，上方自由曳动。

### 上方 Plan（刺绣和音骨架）
`壹` 保续低音上 = `壹(肆2)` → `壹` → `壹(贰)` → `壹`

```
Sop: e'' - c'' - d'' - c''
Alt: g'  - e'  - f#' - e'
Ten: c'  - c'  - c'  - c'
Bas: C   - C   - C   - C
```

### 和音表示（复合和音记法）
| 拍位 | 复合和音 | 说明 |
|------|----------|------|
| 1 | `壹` | T，定位 |
| 2 | `肆2@壹`（或 `壹(肆2)`） | 刺绣和音（=肆的第2转位），壹保续上方 |
| 3 | `贰@壹`（或 `壹(贰)`） | 属上和音，壹保续上方 |
| 4 | `壹` | T，定位还原 |

## 示例四：短调 (a moll) K2 全终止

### 和音进行
`壹` → `贰` → `伍7` → `壹`（a moll: i → ii° → V7 → i）

```
Sop: a'' - f'' - g#' - a''
Alt: e'  - d'' - d'' - c''
Ten: c'  - a'  - b'  - a'
Bas: A   - F   - E   - A
```

> 注：短调中 `贰` 为减 3 和音，原位可用。`伍7` 的导音 g#' 需确保 2 度上行至 a''。

## 示例五：含 `壹2省伍7` 的终止定型变体

使用终止定型变体 (2')：`贰` → `壹2省伍7` → `壹`

### 和音进行
`壹` → `贰` → `壹2省伍7` → `壹`（C dur）

```
Sop: c'' - a'  - b'  - c''
Alt: g'  - f#' - d'  - e'
Ten: e'  - d'  - g   - c'
Bas: C   - F   - E   - C
```

> `壹2省伍7` = `壹` 的第 2 转回位置上的根省属 7 和音。
> 在此语境下，`壹2` 的功能不是 T，而是与 `伍7` 一体的 D。
> Ten：g → c' (5 度下行，重复第 5 音他方进行) 为容许范围。
