# 岛冈让体系和声学 (东京艺术大学)

## 核心哲学：曳动理论

和声的根本视角是**曳动**——和音如同钟摆，在稳定位置（定位）与偏离位置（转位）之间来回摆动。稳定与紧张的交替构成和声的运动本质。

> 记号规则说明：本仓库的纯文本和声记号语法已单独整理为 [references/notation-syntax.md](references/notation-syntax.md)。如需阅读正式语法说明，请先查看该文档。

- **定位**：和音的稳定形态（原音）
- **转位**：构成音的偏离形态，产生紧张感
- **曳动单位**：从松手到回落终止的完整摆动过程（一组特定的和音进行）
- **还原**：将转位音还原为定位音，消解紧张

## 基本术语体系

### 和音记号
- **音度记号**：`壹`、`贰`、`叁`、`肆`、`伍`、`陆`、`柒`（作为音级核心）
- **形体指数**：紧跟在音度记号后，7=七和音, 9=九和音
- **转回指数**：紧跟在数字后，1=第1转位, 2=第2转位, 3=第3转位
- **根省属和音（根省形）**：根音被省略的形体，记作带有 `省` 前缀的形式（如 `省伍7`）

### 低音位
- **基本位置**〈基〉：根音最低
- **第1转回位置**〈1转〉：第3音最低
- **第2转回位置**〈2转〉：第5音最低
- **第3转回位置**〈3转〉：第7音最低

### 功能体系

| 记号 | 名称 | 对应音度 |
|------|------|----------|
| T | 主功能 | `壹` (`陆`作为代理) |
| D (或D1) | 属功能 | `伍` |
| D2 | 第2属功能 | `贰`、`贰7`、`肆` |
| D3~D6 | 第3~6属功能 | `陆`、`叁`、`柒`、`肆` |
| S | 下属功能 | `肆` |

**5度圈**：`壹` $\rightarrow$ `伍` $\rightarrow$ `贰` $\rightarrow$ `陆` $\rightarrow$ `叁` $\rightarrow$ `柒` $\rightarrow$ `肆` $\rightarrow$ `壹`（右回=D进行）

### 常用曳动单位

| 类型 | 进行 | 记号 |
|------|------|------|
| 曳动单位第1型 | T → D → T | `壹` $\rightarrow$ `伍` $\rightarrow$ `壹` |
| 曳动单位第2型 | T → D2 → D → T | `壹` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹` |
| 曳动单位第3型 | T → S → T | `壹` $\rightarrow$ `肆` $\rightarrow$ `壹` |

### 终止类型

| 名称 | 略记 | 和声型 | 功能 |
|------|------|--------|------|
| 全终止 | □ | $\rightarrow$ `伍` $\rightarrow$ `壹` | D $\rightarrow$ T |
| 伪终止 | □̷ | $\rightarrow$ `伍` $\rightarrow$ `陆` | D $\rightarrow$ T(代理) |
| 半终止 | ▷ | $\rightarrow$ `伍` | D |
| 变终止 | □̱ | $\rightarrow$ `肆` $\rightarrow$ `壹` | S $\rightarrow$ T |

**闭终止**：以 `壹` 结束（全终止/变终止）。**开终止**：以不稳定和音结束（伪终止/半终止）。

## 四声体基础规则

### 声部
上から：Soprano (Sop) → Alto (Alt) → Tenor (Ten) → Bass (Bas)
- **外声**：Sop + Bas（对位关系最重要）
- **内声**：Alt + Ten
- **上3声**：Sop + Alt + Ten

### 基本配置规则
1. 可省略的构成音只有**第5音**（根省属和音的根音省略是特殊形体）
2. 第5音可重复，第3音和根音原则上不重复（3和音中根音重复为特殊配置）
3. 邻接声部间距离：8度以内为标准，超过15度为离隔（应避免）
4. 上3声标准配置：**密集配分**（8度以内）或**开放配分**（超过8度）

### 连接规则
1. **共同音保持**：前后和音的共同音尽量在同一声部保持
2. **限定进行音**：导音（第7音=第3音上方半音）必须2度上行到主音；第7音（和音的第7构成音）必须2度下行
3. **禁止**：隐伏8度/5度（外声同向进入完全8度/5度），平行8度/5度
4. **容许**：减5度→完全5度（上行时），反向5度/8度，斜向5度/8度

## 四声体写作步骤

以下步骤适用于任一给定课题（如指定的 Sop 旋律或 Bas 线条）。按此顺序执行可获得符合岛冈让体系规则的 SATB 四声体。

### Step 1: 调性确定与终止配置
- 辨明课题的调性（长调/短调、调位）。
- 确定乐句结构：在何处配置全终止（闭终止），何处配置半终止（开终止）。
- 若课题较长（>4小节），可能需要中间伪终止或变终止作为补充。

### Step 2: 曳动单位骨架选择
- 根据课题的和音分布，选定曳动单位序列。最常用组合：
  - K2（`壹` → `贰` → `伍` → `壹`）：最标准的终止曳动单位
  - K1（`壹` → `伍` → `壹`）：较简短的收束
  - K3（`壹` → `肆` → `壹`）：变终止用途
- 在每个终止点前确保有至少一个完整的曳动单位。

### Step 3: 终止定型确定（关键步骤）
1. 在全终止处：选择终止定型 = D2 → D → T（`贰` → `伍7` → `壹` 等）。
2. 从 [references/d-formation.md](references/d-formation.md) 选取：
   - Sop 定型（1）~（5）：决定外声 Sop 的进行模式
   - Bas 定型（a）或（b）：决定 Bas 的进行模式
3. 在非终止点：可使用 D 定型（g）`伍` → `陆` 作为伪终止基础。

### Step 4: Bas 线条填充
- 从终止定型开始向两端延伸，填充全部 Bas 进行。
- 基本原则：Bas 以 4 度以内进行为主，避免不必要的远跳。
- 在反复进行段落中：按移度/移调规则系统移动 Bas。

### Step 5: Sop 旋律构建
- 在终止定型点按选定 Sop 定型分配 Sop 进行。
- 非终止点：Sop 旋律应力求流畅，优先使用顺次进行（2 度上下），适度使用跳越（3 度以上）。
- Sop 作为最高声部应有旋律完整性，避免过长持续同音。

### Step 6: 内声（Alt + Ten）填充
- 依据上 3 声配置规则，将 Alt 和 Ten 填入：
  - 找出前后两和音的共同音，在同一声部保持。
  - 导音（第 7 音）必须 2 度上行至主音。
  - 第 7 音（7 和音的构成音）必须 2 度下行。
  - 剩余声部向手近音进行。
- 配分一致性：密集 → 密集、开放 → 开放（或 Oct → 密集/开放等近似配分）。

### Step 7: 规则检查
逐条检查 [references/rules.md](references/rules.md) 中的 A-G 条款：
- **A 配置规则**：第 5 音省略/重复是否合规；第 9 音位置是否正确。
- **B 进行规则**：导音/第 7 音的限定进行是否满足；隐伏 5 度/8 度。
- **C 同时进行规则**：平行 8 度/5 度；并达 8 度/5 度；声部交叉。
- **D 低音规则**：Bas 跳进是否过远。
- **E 配置变化规则**：间接连续 5 度/8 度。
- **F/G 装饰规则**（如使用了装饰音/转位）。

### Step 8: 装饰添加（可选）
- 在严恪 Bas 课题中，以课题指定的 Bas 为骨架，Sop 可以添加经和音、刺绣音、侍音、挂留音等转位/装饰。
- 规则 F1~F3、G1~G2 特别需要关注。

## 关键概念速查

### D定型
`伍` $\rightarrow$ `壹` 的固定连接模式，Sop有8种定型，Bas对应不同低音进行。

### 终止定型
全终止 D $\rightarrow$ T 被 D2 和音补强后的形式：D2 $\rightarrow$ D $\rightarrow$ T（`贰` $\rightarrow$ `伍` $\rightarrow$ `壹` 等）。

### VI作为T
`陆` 可在曳动单位的 T 项中作为 `壹` 的代理和音使用，常用 `壹` $\rightarrow$ `陆` 的 T 连用形。

### 转调/借用/转旋
- **转调**：调位变更（主音移动）
- **移旋**：调种变更（长调↔短调）
- **借用**：从他调临时借用和音（标记：`+贰`、`+肆`等）
- **转旋**：功能性的等音转换，不改变主音但改变调的感觉

### 偶成和音
由曳动（转位）偶然形成的和音集合。偶成和音的识别需要还原为原和音。

### 保续低音
`壹` 或 `伍` 音在低音部长期保持，上方声部形成自由曳动的和声进行。

### 反复进行
固定和声模式按一定度数反复上下移动。
- **移度反复**：在同一调内移动
- **移调反复**：每次移动都转调
- **D型**：基于D进行的反复 / **S型**：基于S进行的反复 / **复合型**：复合和声模式的反复

# 纯文本和声记号语法说明

记号发明者孙志贵按：本仓库采用这一套可在普通文本环境中直接输入、检索和协作的和声记号系统，用于替代教材中原本依赖特殊排版符号的记述。其目标是：在不丢失和声功能与形体信息的前提下，让四声体写作、和音分析与教学说明都能以纯文本方式表达，便于大语言模型理解这些知识。

## 1. 设计原则

- 记号以“音级核心 + 前缀 + 后缀”的方式组合。
- 前缀说明和音的性质、功能或形体来源；后缀说明和音的变体、转位、附加音或附加标记。
- 语法尽量保持“先说明音级，再说明形体，再说明转位，再说明附加标记”的顺序。
- 本规则定义的是记号层面的书写方式，不替代和声功能解释本身。

## 2. 基本语法骨架

```text
<记号> ::= <前缀>* <音级> <后缀>*
<前缀> ::= 长 | 短 | 省 | 准 | 拿 | +
<音级> ::= 空 | 壹 | 贰 | 叁 | 肆 | 伍 | 陆 | 柒
<后缀> ::= <附加音后缀> | <变音后缀> | <扩展后缀> | <转位后缀> | <附加标记> | <分组符>
```

## 3. 音级记号

- 空：通配符，表示“当前和音”“当前音级”或“任意和音”，常用于模板、示例或泛化说明。
- 壹：I
- 贰：II
- 叁：III
- 肆：IV
- 伍：V
- 陆：VI
- 柒：VII

### 说明

这些音级记号是整个语法的核心。它们可以单独表示一个和音，也可以与前缀和后缀组合成更精确的和音记号。

## 4. 前缀

前缀置于音级之前，用来说明和音的性质或来源。

- 长：强调为长和音；仅在必要时使用。
- 短：强调为短和音；仅在必要时使用。
- 省：表示根音省略形或第五音省略形，置于音级之前。
- 准：表示从同主短调借来的准固有和音。
- 拿：表示那不勒斯 Ⅱ。
- +：在音级之前，等同于“长”，用作强制调式大音响化（如皮卡第主和音、多利安肆和音的强制大三和弦性质）。

### 示例

- 伍：普通的属和音
- 长伍：被强调为长和音的属和音
- 短伍：被强调为短和音的属和音
- 省伍：根省属和音或第五音省略形
- 准伍：准固有和音
- 拿贰：那不勒斯 Ⅱ
- +肆：被强制性大调化（变长三和音）的肆级

## 5. 后缀

后缀置于音级之后，用来说明和音的变体与附加信息。

### 5.1 附加音后缀

- +4：表示附加第四音
- +6：表示附加第六音

若附加音本身有升降，在 `+` 之后附加修饰符半音：
- +升6：表示附加升六度音（如 `肆+升6`）
- +降6：表示附加降六度音

### 5.2 扩展后缀

- 7：七和音
- 9：九和音
- ~：表示延展、连续或“更宽的和声展开”语义；使用时需视上下文而定

### 5.3 转位后缀

- 0：表示任意转回形或转位形的占位符
- 1、2、3、4：表示具体的转回形或转位形

### 说明

扩展后缀与转位后缀在本体系中应最先书写，并通常按“先扩展，再转位”的顺序排列。例如：

- 伍72：属七的第二转回形
- 伍93：属九的第三转回形
- 伍00：属和音（无论属七还是属九）的任意转回形

### 5.4 变音后缀

- 降5：表示第 5 音下变；置于扩展后缀与转位后缀之后。
- 升5：表示第 5 音上变；置于扩展后缀与转位后缀之后。

> 说明：在本体系里，升5 与 降5 主要用于记录和音内第五音的变音；其书写顺序为“先扩展/转位，再变音”，具体音高解释需结合功能语境决定。

## 6. 附加标记

- @：副次调指定记号，用于标记某个和音所指向的副次调目标。
- /：保续低音指定记号，用于标记某个低音保持不变或被持续强调的情况。

### 说明

这两个标记通常作为附加标记出现在记号尾部，表示“功能上的目标方向”或“低音上的持续性”。其具体内容可以由后续文字补充说明。

## 7. 分组与嵌套

括号与方括号用于包裹复合和声、嵌套结构或需要强调的整体片段。

- ()：普通括号，用于包裹复合记号或说明性片段
- []：方括号，用于包裹次级说明或内部结构
- {}：花括号，用于包裹更复杂的组合关系

## 8. 推荐书写顺序

为了保持可读性和统一性，建议依次按以下顺序书写：

1. 前缀
2. 音级
3. 扩展信息（如 7、9）与转位信息（如 1、2、3）
4. 变音（如 升5、降5）/ 附加音信息（如 +4、+6）
5. 附加标记（如 @、/）
6. 分组符号

### 例子

- 伍：普通的属和音
- 长伍72：强调为长和音的属七第二转回形
- 省伍降5：根省属和音且第 5 音下变
- 准伍+6：同主短调借来的准固有和音并附加第六音
- 伍@：带副次调目标标记的属和音
- 伍/：带保续低音标记的属和音

# 和音体系与功能理论

## 调与音度

- **主音**：调的中心音
- **调位**：以主音音名表示（C, D, E...）
- **调种（旋法）**：长旋法（长音阶）/ 短旋法（短音阶）
- **24调**：12调位 × 2旋法

### 音度
主音为 `壹` 度（I度），顺次上方数 `贰`、`叁`、`肆`、`伍`、`陆`、`柒`。八度关系的音度同度数。

### 音度变位
各音度可在保持同一性的前提下半音变化：
- **上方变位**：`升壹`、`升贰`、... `升柒`（或标记为 `升`）
- **下方变位**：`降壹`、`降贰`、... `降柒`（或标记为 `降`）
- 原位置为**正位**

## 和音基础

### 3和音
由根音(1)、第3音(3)、第5音(5)构成。

原型：根音在最低位置。

### 7和音
3和音原型上加第7音(7)。根音上方7度。

### 低音位
- **基本位置**〈基〉：根音最低
- **第1转回位置**〈1转〉：第3音最低
- **第2转回位置**〈2转〉：第5音最低
- **第3转回位置**〈3转〉：第7音最低

### 高音位
最高音为根音=1高，第3音=3高，第5音=5高，第7音=7高。

### 构成音重复与省略
- 可省略：**第5音**（唯一可省略的构成音）
- 可重复：**第5音**（根省属和音的根音省略为特殊形体）

## 和音的5度圈与功能

### 5度圈（右回 = D进行）
`壹` $\rightarrow$ `伍` $\rightarrow$ `贰` $\rightarrow$ `陆` $\rightarrow$ `叁` $\rightarrow$ `柒` $\rightarrow$ `肆` $\rightarrow$ `壹`

### 功能位置
- **T（主功能）**：`壹` 的位置。稳定，自由移动到任何不稳定和音。
- **D（属功能，D1）**：`伍`。距T 1次D进行。
- **D2~D6**：距T 2~6次D进行的位置。
- **S（下属功能，S1）**：`肆`。S进行（左回）距T 1次。

功能位置由「距T的D进行次数」决定。

### 常用功能
实际最常用：T, D(D1), D2, S。

## 和音记号体系

| 记号 | 名称 |
|------|------|
| `壹`、`贰`、`叁`... | 音度记号（纯文本核心记号） |
| `壹7`、`贰7`、`伍7`... | 7和音 |
| `壹2`、`贰2`、`伍2`... | 第2转回位置 |
| `省伍7`、`省伍9` | 根省属7 / 根省属9（省根形体） |
| `+肆`、`+贰` | 借用和音 |
| `降柒` | 副D2和音（降柒级借用和音，勿与「副V」混淆） |
| `壹@伍` | 相对调性和音 / 复合和音 |
| `拿贰1` | 那不勒斯 Ⅱ |
| `+壹` / `长壹` | 皮卡第主和音 |

## D和音（`伍`上和音）

- `伍`（3和音）
- `伍7`（7和音）
- `伍9`（9和音）
- `省伍7`（根省属7）
- `省伍9`（根省属9）

## D2和音（`贰`上和音）

- `贰`、`贰7`、`肆`（固有）
- `+贰`、`+肆`、`降柒`（借用/变位）
- 根省属各种形体

### 增6和音
`伍7` 的第5音下方变位（降5）形成的特殊 D2 和音：
- **增3-4**（法国增6）
- **增3-6**（意大利增6）
- **增5-6**（德国增6）

## 根省属和音（根省形）

根音被省略的形体。以 `省` 前缀和音级结合表示。

- **省伍7**：`伍7` 的根音省略 $\rightarrow$ 由第3音、第5音、第7音构成
- **省伍9**：`伍9` 的根音省略

根省属和音不是独立和音，是 `伍7`/`伍9` 的省略形体，功能仍属于 D。

配置时因只有3个构成音，4声体中必须重复某音。第5音重复常可能。〈2转〉时第7音重复也常见。

## 副V（下方变位V）

`伍` 的第5音下方变位（降5）形成的和音。短调中常见。
副 `伍7` 仅在 D2（`贰` 上）适用，其他音度上的副 `伍7` 不适用（会引向转调）。

## VI作为T

`陆` 是 D3 位置的不稳定和音，但在 `伍` 后（$\rightarrow$ `陆`）作为 `壹` 的代理和音（T）使用。

- `陆`(T) 只用 〈基〉
- 伪终止 = `伍7` $\rightarrow$ `陆` （D定型 `伍` $\rightarrow$ `壹` 的 `壹` 换成 `陆`）
- `壹` $\rightarrow$ `陆` 的 T 连用：`壹` 常先行于 `陆`
- `陆`(T) 的后续以 D2 和音为主

## S和音（`肆`上和音）

### 5种形体
`肆`（3和音）$\rightarrow$ `肆7` $\rightarrow$ `省肆7` $\rightarrow$ `肆9` $\rightarrow$ `省肆9`，以及 `+肆` 系列（多利安等）。

### S定型
S $\rightarrow$ T 的进行（变终止基础）：`肆` $\rightarrow$ `壹`。

# 四声体配置与连接规则

## 声部

上から：Soprano (Sop) → Alto (Alt) → Tenor (Ten) → Bass (Bas)

- **外声**：Sop + Bas（最重要对位关系）
- **内声**：Alt + Ten
- **上3声**：Sop + Alt + Ten

## 各声部音域

Sop: c¹ ~ a²（点线至c³）
Alt: g ~ f²（点线至a²）
Ten: c ~ c²（点线至f¹）
Bas: F ~ c¹（点线至e¹）

## 邻接声部标准距离

- **密集配分**：上3声中最高与最低在8度以内
- **开放配分**：上3声中最高与最低超过8度
- **离隔**：邻接声部超过15度（应避免）
- **Oct配分**：Sop与Alt或Alt与Ten之间为8度

## 上3声标准配置

各3和音的上3声有4种标准配置：
1. 1高, 3, 5（根音最高）
2. 3高, 1, 5
3. 5高, 1, 3
4. 3高, 5, 1（仅I, IV, V的3和音）

## 连接基本法

### 共同音保持
前后和音的共同音尽量在同一声部保持不动。

### 3声进行
- **顺次进行**：2度上下
- **跳越进行**：3度以上
- **3声法则**：上3声中每个声部各自独立进行，禁止2声部同向同度到达同一音。

### 限定进行音
- **导音**（第7音=第3音上方半音）：必须2度上行到主音
- **第7音**（7和音的第7构成音）：必须2度下行
- 导音重复原则上禁止（`伍7` $\rightarrow$ `壹` 时尤为严格）

## 禁止事项

### 连续8度/5度
- 2声部同向进入完全8度或完全5度
- 上2声间尤其严格
- 外声间最严格

### 隐伏8度/5度
- 外声同向进入完全8度/5度（Sop跳越进行时）

### 并达8度/5度
- 2声部从不同音同向到达同一音（同度）

## 容许事项

- 减5度→完全5度（限定进行音形成的连续5度）
- 反向连续5度/8度
- 斜向连续5度/8度
- 内声间连续5度（较外声宽松）
- 配置变化引起的间接连续5度/8度

## 配置变化

和音音度不变仅配置变化（含形体/低音位变化）。

### D和音形体高次化
3和音→7和音→9和音：无条件可能。

### D和音形体低次化
- 9→3：禁止
- 9→7：容许
- 7→3：禁止

### 配置变化中的禁止
- 第7音→根音的2度上行
- 第9音→第3音的2度上行

### 配置变化中的5度/8度
- 配置变化同时生发的连续5度：先行5度为减5度时，后续完全5度容许
- 配置变化引起的并达5度/8度：两外声间全部良好

# 曳动单位与终止式

## 曳动（ゆれ）

和声的根本视角。和音如钟摆，在稳定位置（定位）与偏离位置（转位）间来回摆动。

- **定位**：稳定形态（原音）
- **转位**：偏离形态，产生紧张感
- **曳动单位**：从松手到回落终止的完整摆动过程
- **曳动连锁**：多个曳动单位串联

## 和音的道筋

不稳定和音到 `壹` (T) 的复归路径：

- T（稳定）自由出发到任何 D1~D6
- D1(`伍`)直接 $\rightarrow$ T
- D2(`贰`)需经 D1 $\rightarrow$ T
- D3(`陆`)需经 D2 $\rightarrow$ D1 $\rightarrow$ T
- D4(`叁`)需经 D3 $\rightarrow$ D2 $\rightarrow$ D1 $\rightarrow$ T
- D5(`柒`)需经 D4 $\rightarrow$ D3 $\rightarrow$ D2 $\rightarrow$ D1 $\rightarrow$ T
- D6(`肆`)需经 D5 $\rightarrow$ D4 $\rightarrow$ D3 $\rightarrow$ D2 $\rightarrow$ D1 $\rightarrow$ T

## 各种曳动单位

### 常用曳动单位

| 名称 | 进行 | 功能 | 略记 |
|------|------|------|------|
| 曳动单位第1型 | T $\rightarrow$ D $\rightarrow$ T | `壹` $\rightarrow$ `伍` $\rightarrow$ `壹` | K1 |
| 曳动单位第2型 | T $\rightarrow$ D2 $\rightarrow$ D $\rightarrow$ T | `壹` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹` | K2 |
| 曳动单位第3型 | T $\rightarrow$ S $\rightarrow$ T | `壹` $\rightarrow$ `肆` $\rightarrow$ `壹` | K3 |

### 非常用曳动单位

| 长度 | 进行示例 |
|------|----------|
| 4项 | `壹` $\rightarrow$ `陆` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹` (T $\rightarrow$ D3 $\rightarrow$ D2 $\rightarrow$ D $\rightarrow$ T) |
| 5项 | `壹` $\rightarrow$ `叁` $\rightarrow$ `陆` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹` |
| 6项 | `壹` $\rightarrow$ `柒` $\rightarrow$ `叁` $\rightarrow$ `陆` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹` |
| 7项 | `壹` $\rightarrow$ `肆` $\rightarrow$ `柒` $\rightarrow$ `叁` $\rightarrow$ `陆` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹` |
| 8项 | (最长，反复进行形中可见) |

短曳动单位使用频率高。T最高，D1次之，越远离T越低。

## 终止类型

### 基本终止

| 名称 | 略记 | 和声型 | 功能 | 句读感 |
|------|------|--------|------|--------|
| 全终止 | □ | $\rightarrow$ `伍7` $\rightarrow$ `壹` | D $\rightarrow$ T | 句号(.) |
| 伪终止 | □̷ | $\rightarrow$ `伍7` $\rightarrow$ `陆` | D $\rightarrow$ T(代理) | 分号(;) |
| 半终止 | ▷ | $\rightarrow$ `伍` | D | 逗号(,) |
| 变终止 | □̱ | $\rightarrow$ `肆` $\rightarrow$ `壹` | S $\rightarrow$ T | 句号(.) |

### 其他终止

| 名称 | 和声型 |
|------|--------|
| 不完全终止 | 非 `伍` 的不稳定和音终止 |
| 代理终止 | V代理（如 `贰`）终止 |

### 闭终止与开终止

- **闭终止**：以 `壹` 结束（全终止/变终止）。段落完全稳定。
- **开终止**：以不稳定和音结束（伪终止/半终止）。段落未完结。

## 终止定型

全终止的 D $\rightarrow$ T 被 D2 和音补强后的形式：

**D2 $\rightarrow$ D $\rightarrow$ T**（如 `贰` $\rightarrow$ `伍` $\rightarrow$ `壹`）

终止定型是古典音乐中最常用的终止形式。

### 终止定型的D2项
可用和音：`贰`、`贰7`、`肆`、`省贰7`、`+贰`、`+肆`等。

### 终止定型Sop定型

| 定型 | Sop进行 | 说明 |
|------|---------|------|
| (1) | `贰` `伍7` `壹` | 基本形 |
| (2) | `贰7` `伍` `壹` | 带7和音 |
| (3) | `贰7` `伍7` `壹` | 双7 |
| (4) | `肆` `伍7` `壹` | IV型 |
| (5) | `贰` `伍` `壹` | `伍`省略7 |

### 终止定型Bas定型
Bas在 `贰` $\rightarrow$ `伍` 通常为4度上行（`贰` $\rightarrow$ `伍`），5度下行少用。

### `壹2省伍7` 在终止定型中的应用

`壹2省伍7` 的 `壹2` 重复形与 `伍7` 连接时，`壹2` 的功能不是 T 而是与 `伍7` 一体的 D。
`壹2省伍7` 遵循强 $\rightarrow$ 弱关系，多在强部 $\rightarrow$ 弱部使用。

### 导音例外进行

为避免与挂留音冲突，导音可向 5 或 3 例外进行（非上行到 1）。Sop 的导音下行应避免。

## 曳动单位连锁

多个曳动单位串联，结节点在 `壹` (T)：

例：K3 + K1 + K2 = `壹` $\rightarrow$ `肆` $\rightarrow$ `壹` $\rightarrow$ `伍` $\rightarrow$ `壹` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹`

## 常用终止型组合

### 曲尾终止
- 最常用：全终止（终止定型）
- 变终止补充：全终止后附加变终止
- 最终乐段：全终止或伪终止均可

### 中间终止
- 乐句终止点 `壹` $\rightarrow$ 伪终止
- 半终止：乐句中间段落

### 终止定型前D2连用
终止定型前 D2 和音可连用：`贰` $\rightarrow$ `贰7` $\rightarrow$ `伍7` $\rightarrow$ `壹`

# D定型与终止定型

## D定型

`伍7` $\rightarrow$ `壹` 的固定连接模式。

### Bas D定型（7种）

| 定型 | Bas进行 | 说明 |
|------|---------|------|
| (a) | `伍7` $\rightarrow$ `壹`（下行4度） | 最基本 |
| (b) | `伍7` $\rightarrow$ `壹`（上行2度） | 同度Bas |
| (c) | `伍7` $\rightarrow$ `壹`（上行4度） | 上行跳进 |
| (d) | `省伍7` $\rightarrow$ `壹` | 根省属 |
| (e) | `伍7` $\rightarrow$ `壹2`（下行2度） | 到达第2转回 |
| (f) | `伍` $\rightarrow$ `壹` | 3和音伍 |
| (g) | `伍` $\rightarrow$ `陆` | D定型补充（伪终止） |

### Sop D定型（8种）

| 定型 | Sop进行 |
|------|---------|
| (1) | `伍7` $\rightarrow$ `壹` $\rightarrow$ `壹` |
| (2) | `伍7` $\rightarrow$ `壹` $\rightarrow$ 3 |
| (3) | `伍7` $\rightarrow$ 3 $\rightarrow$ `壹` |
| (4) | `伍7` $\rightarrow$ 3 $\rightarrow$ 3 |
| (5) | `伍7` $\rightarrow$ 5 $\rightarrow$ `壹` |
| (6) | `伍` $\rightarrow$ `壹` $\rightarrow$ `壹` |
| (7) | `伍` $\rightarrow$ `壹` $\rightarrow$ 3 |
| (8) | `伍` $\rightarrow$ 3 $\rightarrow$ `壹` |

### D定型连接规则
- `伍7`〈基〉$\rightarrow$ `壹`〈基〉：上3声密集 $\rightarrow$ 密集 或 开放 $\rightarrow$ 开放（一致/近似）
- 导音2度上行（必须）
- 第7音2度下行（必须）
- `伍7`不完全形 $\rightarrow$ `壹`标准外配置（第5音欠如）

### `伍7` $\rightarrow$ `壹2`（〈2转〉）的特殊处理
- 第7音在3下方时：第7音可2度上行（例外）
- 第7音在3上方时：第7音2度上行会生连续5度，不可。`壹2` 第3音重复（标准外配置）为容许

### D定型(g) `伍` $\rightarrow$ `陆`
- 伪终止的基础
- 上3声：导音2度上行，第7音2度下行，第5音/根音各自向手近音下行
- `陆` 上3声：根音欠，第3音重复
- `伍7` 必完全形，不完全形不用

## 终止定型

### 终止定型结构

**D2 $\rightarrow$ D $\rightarrow$ T**（终止定型的基本型）

D $\rightarrow$ T 部分与 D 定型相同。D2 $\rightarrow$ D 的连接是关键。

### D2 $\rightarrow$ D连接

| 进行 | 配分规则 |
|------|----------|
| `贰` $\rightarrow$ `伍7` | 一致（密 $\rightarrow$ 密，开 $\rightarrow$ 开）或近似（Oct $\rightarrow$ 密，Oct $\rightarrow$ 开） |
| `贰7` $\rightarrow$ `伍7` | 同上 |
| `肆` $\rightarrow$ `伍` | 同上 |

### 终止定型Sop定型

与D定型Sop定型对应：

| 定型 | Sop进行 | D2项 |
|------|---------|------|
| (1) | `贰` `伍7` `壹` | `贰`〈基〉 |
| (2) | `贰7` `伍` `壹` | `贰7`〈基〉 |
| (3) | `贰7` `伍7` `壹` | `贰7`〈基〉 |
| (4) | `肆` `伍7` `壹` | `肆`〈基〉 |
| (5) | `贰` `伍` `壹` | `贰`〈基〉 |

加上两外声组合终止定型(5种)和Bas定型(2种)的组合，形成各种实际形态。

### 终止定型中 `壹2省伍7` 的应用

`壹2省伍7`(D)作为终止定型的D项使用时，Sop定型变为(2')~(5')：

| 定型 | 变化 |
|------|------|
| (2') | `贰` `壹2省伍7` `壹` |
| (3') | `贰7` `壹2省伍7` `壹` |
| (4') | `贰7` `壹2省伍7` `壹`（同3'） |
| (5') | `肆` `壹2省伍7` `壹` |

`壹2省伍7` 与 `壹2` $\rightarrow$ `伍7` 连用时，`壹2` 的功能不是 T 而是与 `伍7` 一体的 D。

### 终止定型规则
- D2 $\rightarrow$ D配分：一致或近似
- `壹2省伍7`(D) 连接：两和音配分一致
- D2 $\rightarrow$ `壹2省伍7`(D) 连接：配分一致或近似
- 开放配分的 `贰` 用于定型(5')时，内声间生连续5度，不可。此定型只用密集 `贰`

### 终止定型Bas定型

| 定型 | Bas进行 |
|------|---------|
| (a) | `贰` $\rightarrow$ `伍` $\rightarrow$ `壹`（上行4度 $\rightarrow$ 下行4度） |
| (b) | `贰` $\rightarrow$ `伍` $\rightarrow$ `壹`（同度 $\rightarrow$ 下行4度） |

### 两外声组合终止定型

Bas定型(a)(b) × Sop定型(1)~(5) 的组合，共10种基本组合。

### 半终止点的 `壹2省伍7`

半终止点 `伍` 也可用 `壹2省伍7`：
- `壹2省伍7` $\rightarrow$ `伍` 连用时，`壹2` 为D功能，强 $\rightarrow$ 弱关系
- 半终止外 `伍` $\rightarrow$ `壹2省伍7` 也遵循强 $\rightarrow$ 弱

# 转调/借用/转旋

## 各音度调

某调X的 `壹`、`贰`、`叁`、`肆`、`伍`、`陆`、`柒` 各3和音分别作为 `壹` 的调：

- `壹` 调 = 主调自身
- `伍` 调 = 属调
- `贰` 调 = 上主调
- 等

## 转调

乐曲进行中调发生变化。

### 近亲转调
- 主调的 `伍` 调（属调）
- 主调的 `贰` 调（上主调）
- 主调的 `陆` 调（下中调）

### 中介调转调
通过中介调进行的转调。如 C dur $\rightarrow$ a moll $\rightarrow$ C dur。

### 转调方法
- 共同和音转调
- 转义转调
- 等音转调

## 借用

从他调临时借用和音。

### 常用借用
- **`+肆`（大 `肆`）**：短调中借用长调的 `肆`
- **`+贰`（大 `贰`）**：短调中借用长调的 `贰`
- **`降柒`**：长调中借用短调的下主音
- **`拿贰`（那不勒斯 `贰`）**：降 `贰` 级的大3和音第1转回（`拿贰1`）

### 借用标记
`+贰` = 短调借用长调的 `贰`（大3和音）
`+肆` = 短调借用长调的 `肆`（大3和音）
`+贰7` = 短调借用长调的 `贰7`
`+肆7` = 短调借用长调的 `肆7`

## 转旋

功能性的等音转换。不改变主音但改变调的感觉。

转旋通过等音理解和音的功能转换。

## 异名同音的转调

利用异名同音（同音异名）进行的转调。

### 减7的异名同音转义
减7各构成音间距离均为3半音，1个减7可有4种异名同音把握。

### 等音转调步骤
1. 目标调的减7 = 原调的减7（异名同音转义）
2. 从该减7解决到目标调 `壹`

## 转调+转旋的调关系扩大

通过转调和转旋的组合，可到达远关系调。

### D圈调关系
D1(属调) $\rightarrow$ D2(上主调) $\rightarrow$ D3(下中调) $\rightarrow$ ... 逐步扩大可达范围。

### S圈调关系
S1(下属调) $\rightarrow$ S2(下主调) $\rightarrow$ S3(中调) $\rightarrow$ ...

## 转调连接规则

转调时的连接遵循普通连接规则，但：
- 可容许增2度进行（转调连接中的特殊处理）
- 导音在新调中仍需2度上行到新调主音
- 转调点附近避免限定进行音的冲突

# 各种D和音与D2和音

## D和音一览（长/短两调）

| 类型 | 3和音 | 7和音 | 根省属7 | 9和音 | 根省属9 |
|------|-------|-------|---------|-------|---------|
| 固有伍 | `伍` | `伍7` | `省伍7` | `伍9` | `省伍9` |
| 副伍 | `降伍` | `降伍7` | `省降伍7` | `降伍9` | `省降伍9` |
| 借用+肆 | — | `+肆7` | `省+肆7` | `+肆9` | `省+肆9` |

## `伍9` 和 `省伍9`

### `伍9` 配置
- 长9音/短9音：根音上方9度以上
- 长9音必须在第3音（导音）上方
- 有预备时长9音可在第3音下方
- 短9音可在第3音下方

### `伍9` 解决
- 第9音：2度下行
- 第7音：2度下行
- 导音：2度上行

### `省伍9` 配置
根省属9只有3种构成音，4声体中必须重复。第5音重复常可能，〈2转〉时第7音重复也常见（上2声重复除外）。

### `省伍9` 解决
- 第7音、导音限定进行同 `伍7`
- 重复第5音一方2度下行，他方5度下行或4度上行
- 〈2转〉非重复时：第7音下方者2度上行可
- 第7音重复：一方2度下行，他方（第3音下方）2度上行

## `伍7` 与 `省伍7`

### `伍7` 配置
完全形（4个不同构成音）和不完全形（第5音欠如，第3音重复）。

- `伍7`〈基〉$\rightarrow$ `壹`：`壹` 必为标准外配置（第5音欠如）
- `伍7`〈1转〉：各种可能
- `伍7`〈2转〉：各种可能
- `伍7`〈3转〉：各种可能

### `省伍7` 配置
- 〈基〉：第3音、第5音、第7音
- 〈1转〉：同上转位
- 〈2转〉：同上转位（第7音重复常见）
- 〈3转〉：同上转位

### `伍7`/`省伍7` 的 Sop 定型

| 定型 | 说明 |
|------|------|
| (a) | 基本两外声定型 |
| (b) | `伍7`（长9）特有多余定型 |
| (c)-(d) | 各种高音位变化 |

第9音下方有第5音时：第5音2度下行会生连续5度，需改用2度上行或4度上行。

### 导音重复
`伍7` $\rightarrow$ `壹`（`壹` 的3和音）时导音重复禁。
`伍7` $\rightarrow$ `壹7` 时导音重复许。

## D2和音（`贰`上）一览

| 类型 | 3和音 | 7和音 | 根省属7 | 9和音 | 根省属9 |
|------|-------|-------|---------|-------|---------|

## 增6和音

`伍7` 的第5音下方变位（降5）形成的特殊 D2 和音。

### 增6的3种形体
- **增3-4**（法国增6）
- **增3-6**（意大利增6）
- **增5-6**（德国增6）

### 增6配置
- 〈2转〉最常用
- 下方变位音（`省伍` 的第5音降）与上方第3音（T/`肆` 的3）形成增6度

### 增6用法
- 终止定型中：`贰` 处的增6 $\rightarrow$ `伍`
- 变位和音的基本仅适用于 D2（`贰` 上）
- `伍` 以外副 `伍7` 不适用（引向转调）

## D2和音在终止定型外的用法

### 用法A（4音D2定型）
`壹` $\rightarrow$ `贰` $\rightarrow$ `贰` $\rightarrow$ `伍` $\rightarrow$ `壹`
Bas固定，D项可用 `伍7` 各种形体。

### 用法B（半音阶经和音）
`壹` $\rightarrow$ `陆` $\rightarrow$ `贰` $\rightarrow$ `壹`：D2 $\rightarrow$ D间插入半音阶经和音。
可用 `+肆` 代替 `贰`。

## D2定型全可能性

D2 $\rightarrow$ D两和音的低音，原则可用 `贰` 上9和音与 `伍` 上9和音的全构成音（含变位）。

### D2定型一览要点
- `贰`（或其降音）可来自各音度
- `伍`（或其降音）可来自各音度
- 组合中排除限定进行音冲突、第7音重复冲突、规则C5冲突、上声部连续5度

### 短调D2定型
- 固有 `柒` 两种：固有 `柒` + 上变 `柒`（`升柒` = 导音）
- D定型/终止定型 `伍` 处用 `升柒`
- 其他固有和音（`贰7`、`叁7` 等）用固有 `柒`
- `贰7` 第7音（`柒`）是固有 `柒`，其预备音（`伍`）也用固有 `柒`

## S定型

### `肆` $\rightarrow$ `壹`（S $\rightarrow$ T）

### 常用S连用

### `肆`上方变位

### `肆`下方变位

# 转位与装饰

## 转位（ゆれ）

构成音在保持同一性的前提下偏离定位位置。

### 还原
将转位音还原为定位音。

### 转位形态

#### 经和音（カ）
不同高度的2个定位音间音阶级连接的转位音。

#### 邻和音
单个定位音被邻接音度包围的转位音。

**1. 刺绣音（シ）**
前后被2个定位音夹住的邻音。
- 上部刺绣音 / 下部刺绣音

**2. 侍音（イ）**
始定位音省略，直接从转位音开始。
- 上部侍音 / 下部侍音（常用上方变位）

**3. 挂留音（ケ）**
侍音在前和音预备并延长的形态。
- 上部挂留音 / 下部挂留音（下挂较少用）

**4. 逸音（ツ）**
侍音的逆，后方定位音省略。
- 上部逸音 / 下部逸音

**5. 先取音（セ）**
逸音在次和音重新打直的形态。
- 上部先取音 / 下部先取音

### 转位音的解决
邻音还原为定位音，经和音到达定位音。
解决 = 转位引起紧张的消解。

### 连续转位
2个以上转位音连续使用。

常见：连续刺绣音、连续侍音。

### 半音阶经和音（半カ）
长2度2定位音间半音阶连接。用变位音(↑↓)表示。

### 2次转位
转位音进一步转位。
- 上方转位音的上方转位 = T²
- 上方转位音的下方转位 = T↓²
- 下方转位音的下方转位 = ↓²
- 下方转位音的上方转位 = ↓T²

## 装饰

原音向不同构成音跳躍进行（非转位的邻接音度进行）。

### 转位 vs 装饰
- 转位：原音→邻接音度（顺次进行），种类不变仅位置变
- 装饰：原音→其他构成音（跳躍进行），音本身变化

### 装饰还原
装饰音可还原为原音。

## 装饰规则

### 配置规则
1. 侍音/挂留音/刺绣音与原音（同名音）2声部间2度同时音程不可（规则F1）
2. 侍音/挂留音上方声部不可同时有其原音（规则F2）
3. 侍音/挂留音下方声部同时有其原音非误，但Bas以外应避免
4. 装饰音所在拍点外的构成音欠如/声部离隔无妨
5. 装饰音部分的限定进行音重复、第9音与根音关系禁则仍适用（规则F3）

### 进行规则
- 含装饰音时规则B1限制不直接适用
- 7度、9度、各种增音程均容许

### 同时进行规则
1. 含转位音的连续5度/8度原则禁，但含侍音的后续5度可（规则G1）
2. 含装饰音的反行连续5度/8度、并达5度/8度容许（规则G1附则）
3. 原和声中的并行连续5度/8度，装饰后不化也禁。但后续5度斜达则可（规则G2）
4. 原和声中的连续5度/8度、并达5度/8度，后续被侍音间接化则全可（规则G2附则1）
5. 原和声中的反行连续5度/8度，装饰后后续间接化则可（规则G2附则2）
6. 原和声中的并达5度/8度，装饰后后续间接化则可（规则G2附则3）

## 配置变化

和音音度不变仅配置变化（形体/低音位变化）。

### D和音形体变化
- 高次化：3→7→9 无条件可能
- 低次化：9→3禁，9→7许，7→3禁

### 配置变化同时的连续5度
先行5度为减5度时，后续完全5度容许。

### 间接连续5度/8度
配置变化产生的间接连续5度/8度：
仅当连续完全8度/5度发生在2个和音内且第2个完全8度/5度在和音交替点时禁（规则E2）。

## 自由Sop课题

### 特征
1. Sop含转位/装饰
2. 下3声装饰控え目
3. 所有转位种类可用（经和音/刺绣音/侍音多用）
4. 和音/调无特别限制
5. 以古典派以后样式为准

### 与严格Bas课题区别
- 自由Sop课题：Sop主役，装饰自由，和音无限制
- 严格Bas课题：Bas主役，对位法样式，规则更严

# 反复进行

## 反复进行定义

固定和声模式每次按一定度数上下移动后反复。

- **反复句**：被反复的和声模式
- **移动距离**：反复句移动的度数
- **移动方向**：上方/下方

## 反复进行型

### 按反复句内容分
- **D型**：反复句 = D进行
- **S型**：反复句 = S进行
- **复合型**：反复句 = 复合和声

### 按移动度数分
- 2度上行 / 2度下行
- 3度上行 / 3度下行
- 等

### 综合标记
D2°上 = D型2度上行，S3°下 = S型3度下行，复3°上 = 复合型3度上行

### 常用反复型（★= 特多用）

**D型**：
- ★ D2度上行型：`壹` `肆` `贰` `伍` `叁`...（移度/移调均可）
- ★ D2度下行型：`伍` `壹` `陆` `肆`...（移度/移调均可）
- D3度上行型：`壹` `叁` `贰` `肆`...
- D3度下行型：`伍` `叁` `壹` `陆`...

**S型**：
- S2度下行型：`壹` `陆` `肆` `贰`...
- ★ S3度下行型：`壹` `陆` `肆`...

**复合型**：
- ★ 复合2度上行型：终止定型风复合
- ★ 复合2度下行型：同上反向
- 复合3度上行型：含 `贰7` 的反复
- 复合3度下行型：同上反向

## 移动方式

### 移度反复
同一调内移动，使用固有音相关。
- 途中可能遇增4度/对斜，容许

### 移调反复
每次都转调，完全保持原D进行形态。

## 反复定型实施

### 各和音配分
反复定型内各和音配分：一致或近似。

### 完结方式
反复进行最终需完结到主和音：
`伍7` $\rightarrow$ `壹(7)` 或 `壹` 终定

### `伍7` 连用配置
〈基〉7和音连用：完全形/不完全形交替。
最初7和音配置决定后续交替。

### 短调反复进行

#### 两种 `柒`
- **固有 `柒`**：短调固有（短7度）
- **上变 `柒`（`升柒`）**：导音

#### 使用分別
1. D定型/终止定型 `伍` 处：用 `升柒`（导音）
2. 其他固有和音（`贰7`、`叁7`等）：用固有 `柒`
3. `贰7` 第7音（`柒`）预备音：用固有 `柒`

## 副 `伍7` 反复进行

各反复句构成：副 `伍7` $\rightarrow$ 固有3和音（各音度调D定型）或 副 `伍7` $\rightarrow$ 副 `伍7`（副 `伍7` 连用）。

副 `伍7` 连用与固有 `伍7` 连用同规则。

## 模仿反复进行

反复进行内形成声部间模仿。

### `伍7` 第7音装饰的模仿
各 `伍7` 第7音装饰后，形成的旋律型在上2声间产生模仿。

### 第7音装饰注意事项
- `伍7` $\rightarrow$ `壹`（`壹` 的3和音）时导音重复禁
- `伍7` $\rightarrow$ `壹7` 时导音重复许
- 副 `伍7` 装饰中不可用导音（第3音）

### 装饰型多样性
固有 `伍7` 可用装饰型多，副 `伍7` 因不可用导音而较少。

## 挂留音补则

### 挂留音配置
侍音同规则适用。

### 导音例外进行
`伍7` $\rightarrow$ `壹` 时到达的主音(1)与他声部挂留音(T)冲突时：
- 导音可向5或3例外进行（许）
- Sop导音下行应避免
- 导音装饰可覆盖不规则动き

### 同时挂留
导音和第5音同时挂留可回避冲突，但外声间同时挂留应避免。

### `伍` $\rightarrow$ `陆` 的导音例外
仅在密集配分的Alt中许（`柒` $\rightarrow$ `陆` 进行）。

# 偶成和音与保续低音

## 偶成和音

由曳动（转位）偶然形成的和音集合。非独立和音，需还原为原和音理解。

### 偶成和音识别
1. 还原转位音为定位音
2. 判断原和音
3. 偶成和音 = 转位过程中临时形成的音集合

### `伍7` 的偶成
`伍7` 各低音位的刺绣音/侍音产生偶成：
- 几乎全部为减7形

### 各种 `伍7` 偶成和音
- `伍7`（各低音位）的刺绣音/侍音
- `伍7` 上方/下方的各种转位组合

## 保续低音

### 定义
`壹` 或 `伍` 音在低音部长期保持，上方声部形成自由曳动的和声进行。

### 保续低音 `壹`
`壹` 音保持，上方形成上部和声。

### 保续低音 `伍`
`伍` 音保持，上方形成上部和声。

### 记法
上部和音记号放入括号 ( )，保续低音音度记在左下小字。

例：`壹(伍7)` = `壹` 保续低音上的 `伍7`

上部和音原则上不附转回指数（实际低音是保续低音）。

### 上部和声的Plan

#### Plan 1: 刺绣和音及其装饰
最简Plan：刺繍和音（`壹` 保续上=`肆2`，`伍` 保续上=`壹2`）为骨架，经和音等装饰扩大。

#### Plan 2: 曳动单位及其连锁
用曳动单位 T D2 D T 或其连锁形成稍发展的形态。

#### Plan 3: 反复进行
保续低音上常用反复进行。

## 复合和音

保续低音短小时，去掉括号，保续低音音度记在下，上部和音记在上。

### 记法
`壹(伍7)` = `壹上伍7` = `壹` 度上的 `伍7`

### 常用复合和音
- `壹` 度上的 `伍7`（`壹` 上 `伍7`）：最常用
- `壹` 度上的 `贰`（`壹` 上 `贰`）
- `伍` 度上的 `贰`（`伍` 上 `贰`）
- `伍` 度上的 `伍7`（`伍` 上 `伍7`）

### 各音度上的副 `伍7`
`壹` 度上 `伍7` 的形可适用于副 `伍7` 的解决和音：
- `贰` 度上的 `伍7` = `贰` 上 `伍7`
- `叁` 度上的 `伍7` = `叁` 上 `伍7`
- 等

### 完全复合和音
`壹` 度上的 `伍7`（`壹伍7`）等，保续低音与上部和音完全复合。

## 同时保续低音

`壹` 和 `伍` 在下2声同时保续。多声部（5声部以上）场合。

同时保续低音 = `壹` 的根音+第5音形成的静止部分。

记法中音度数字2重（如 `壹[伍]`）。

### 保续 `壹`
`壹[伍]` 所有构成音(1,3,5)同时保续 = 保续3和音。

## `壹`/`伍` 以外的保续低音

偶成中静止低音可为 `壹`/`伍` 以外。

- `叁` 保续低音 = `贰`(T)的扩大/延长
- `肆` 保续低音 = `贰7`(D2)的扩大/延长

### 复合和音把握
`壹`/`伍` 以外保续低音较短小时，去括号为复合和音：
- `贰(驹壹)` = `贰` 上 `壹`
- `伍(壹伍7)` = `伍` 上 `肆7`

## 偶成和音的用法

### 刺繍和音
保续低音/反复进行中偶成的刺繍形态和音。

### 侍和音
侍音形成的偶成和音集合。

### 经和音
经和音形成的偶成和音集合。

### 复合偶成
多种转位组合形成的复杂偶成。

## 偶成和音分析步骤
1. 识别低音是否为保续/定位
2. 还原上声部转位音为定位
3. 确定原和音
4. 偶成和音 = 转位过程中的临时音集合

# 四声体学习规则汇总

## A. 配置规则

### A1: 可省略音
第5音可省略。根音省略为特殊形体（根省属和音）。

### A2: 可重复音
第5音可重复。根音/第3音原则上不重复（3和音根音重复为特殊配置）。
根省属配置时第5音重复常可能。

### A2附则: 〈2转〉根省属配置
〈2转〉根省属配置中第7音重复也多（上2声重复除外）。

### A3: 第9音位置
第9音(长/短)必须在根音9度以上上方。

### A4: 长9音位置
长9音必须在第3音(导音)上方。有预备时可在第3音下方。

### A4附则: 有预备长9音
有预备时长9音可在第3音下方。

### A5: 9和音低音位
〈4转〉(第4转回)不用（第9音在低音时9度条件难满足）。

## B. 进行规则

### B1: 声部进行
7度、9度、增音程原则禁（配置变化/装饰音时例外）。

### B1附则: 转调连接
转调连接中增2度可许。

### B2: 限定进行音
- 导音(第7音): 2度上行(必须)
- 第7音: 2度下行(必须)

### B2附则: 例外
- `省伍7` $\rightarrow$ `壹2`：第7音可2度上行(避免 `壹2` 第3音重复)
- 导音可向5/3例外进行(避免挂留音冲突)
- `伍` $\rightarrow$ `陆`：Alt密集时导音可下行到 `陆`

### B3: 隐伏5度/8度
外声同向进入完全5度/8度(Sop跳越时)。

### B3附则: 反复进行
反复进行途中增4度/对斜可许。

## C. 同时进行规则

### C1: 平行8度/5度
2声部同向进入完全8度/5度禁。

### C2: 减5→完5
先行5度为减5度时，后续完全5度可许（上2声间例外）。

### C2附则1: 根音跳进连续5度
先行低音根音跳进形成的连续5度可许。

### C2附则2: 内声连续5度
内声间连续5度较宽松可许。

### C3: 并达8度/5度
2声部从不同音同向到同度禁。

### C3附则: 配置变化并达
配置变化引起的并达5度/8度两外声间全可。

### C4: 反向5度/8度
反向进入完全5度/8度可许。

### C5: 交叉
声部交叉禁（Sop<Alt, Alt<Ten, Ten<Bas）。

## D. 低音规则

### D1: 低音进行
Bas以4度以内进行为原则。

### D2: 低音跳进
9和音低音无预备4度跳进避。

## E. 配置变化规则

### E1: 配置变化禁止
配置变化时：
- 第7音→根音2度上行禁
- 第9音→第3音2度上行禁

### E2: 间接连续5度/8度
配置变化产生的间接连续完全5度/8度：
仅当连续完全8度/5度在2和音内且第2个完全8度/5度在和音交替点时禁。

## F. 装饰规则

### F1: 侍音/刺绣音配置
侍音/挂留音/刺绣音与原音2声部间2度同时音程不可。

### F2: 侍音上方
侍音/挂留音上方声部不可同时有其原音。

### F3: 装饰音限定音
装饰音部分限定进行音重复、第9音与根音关系禁则仍适用。

## G. 装饰同时进行规则

### G1: 转位音连续5度
含转位音连续5度/8度原则禁，但后续5度含侍音则可。

### G1附则: 装饰音连续度
含装饰音反行连续5度/8度、并达5度/8度可许。

### G2: 原和声连续度
原和声中并行连续5度/8度，装饰后不化也禁。后续斜达则可。

### G2附则1: 侍音间接化
原和声中连续5度/8度、并达5度/8度，后续侍音间接化则全可。

### G2附则2: 反行间接化
原和声中反行连续5度/8度，装饰后后续间接化则可。

### G2附则3: 并达间接化
原和声中并达5度/8度，装饰后后续间接化则可。

# SATB 四声体写作示例

下列示例展示从曳动单位选择到具体四声部实现的全流程，可作为 LLM 执行「四声体写作步骤」时的具体参考。

## 示例一：C dur 终止乐句

### 课题设定
- 调性：C dur（长调）
- 长度：4 小节（一个完整曳动单位）
- 要求：以全终止（闭终止）结束

---

### Step 1 & 2: 调性确定 + 曳动单位选择
选择最标准的 K2 型曳动单位：
**T → D2 → D → T** = `壹` → `贰` → `伍` → `壹`

### Step 3: 终止定型确定
- Sop 定型：(1) — `贰` `伍7` `壹`（基本形）
- Bas 定型：(a) — `贰` → `伍` → `壹`（上行 4 度 → 下行 4 度）
- D2项：`贰`〈基〉

### Step 4 & 5: Bas 线条 + Sop 旋律

```
Sop: c'' - a'  - b'  - c''
Alt: g'  - f#' - g'  - e'
Ten: e'  - d'  - d'  - c'
Bas: C   - F   - G   - C
```

### 和音表示

| 拍位 | 和音 | 低音位 | 备注 |
|------|------|--------|------|
| 1 | `壹`〈基〉| 原位 | T，起点，两外声 8 度（密集配分） |
| 2 | `贰7`〈基〉| 原位 | D2，Alt 导音 f#'（升4度） |
| 3 | `伍7`〈基〉| 原位 | D，Ten 第 7 音 d' → c'（2 度下行） |
| 4 | `壹`〈基〉| 原位 | T，终点（全终止），闭终止 |

---

### 规则检查还原

| 规则条款 | 检查内容 | 结果 |
|----------|----------|------|
| B2 | 导音 f#'（Alt，第 3 拍）→ g'（Alt，第 4 拍）：2 度上行 | ✓ |
| B2 | 第 7 音 d'（Ten，第 3 拍）→ c'（Ten，第 4 拍）：2 度下行 | ✓ |
| C1 | 平行 8 度/5 度检查 | ✓ 无 |
| C3 | 并达 8 度/5 度检查 | ✓ 无 |
| C5 | 声部交叉 (Sop > Alt > Ten > Bas) | ✓ 无 |
| A1 | 第 5 音省略 | ✓ 未省略 |
| 共同音保持 | 第 2→3 拍：Alt f#'→g'（2 度上行），Ten d'→d'（保持）| ✓ |

## 示例二：同一乐句的 D 定型表形式

使用 `notation-syntax.md` 的纯文本记号书写：
- 进行：`壹` → `贰7` → `伍7` → `壹`（K2 全终止）
- Sop 定型：(1) `贰` `伍7` `壹`
- Bas 定型：(a) `贰` → `伍` → `壹`（上 4 → 下 4）

四声部（C dur）：
```
Sop: 壹(1高) → 贰7(3高) → 伍7(5高) → 壹(1高)
Alt: 壹(5)   → 贰7(1)   → 伍7(3)   → 壹(3)
Ten: 壹(3)   → 贰7(7)   → 伍7(1)   → 壹(5)
Bas: 壹(1)   → 贰7(1)   → 伍7(1)   → 壹(1)
```

> 说明：上述「(1高)(3高)(5高)(7高)」指该和音中 Sop 所担任的构成音身份；
> 「(1)(3)(5)(7)」指 Bas 对应担任的构成音身份。例如 `贰7(3高)` 表示在 `贰7` 和音中，
> Sop 唱第 3 音（f#'），Bas 唱根音（F）。

## 示例五：含 `壹2省伍7` 的终止定型变体

使用终止定型变体 (2')：`贰` → `壹2省伍7` → `壹`

### 和音进行
`壹` → `贰` → `壹2省伍7` → `壹`（C dur）

```
Sop: c'' - a'  - b'  - c''
Alt: g'  - f#' - d'  - e'
Ten: e'  - d'  - g   - c'
Bas: C   - F   - E   - C
```

> `壹2省伍7` = `壹` 的第 2 转回位置上的根省属 7 和音。
> 在此语境下，`壹2` 的功能不是 T，而是与 `伍7` 一体的 D。
> Ten：g → c' (5 度下行，重复第 5 音他方进行) 为容许范围。
